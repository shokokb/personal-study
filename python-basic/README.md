# 基本事項

* トリプルダブルクオート(""")でヒアドキュメントできる
* バックスラッシュ(\)で改行できる

# 型

* strはstringの省略形
* bool型はTrue or False（先頭が大文字）
* NoneType.None

# 四則演算

* オペランド（非演算子）
* 累乗: **  (2 ** 3 = 8)
* 割り算: // (13 // 8 = 5)
* 余り: % (14 % 4)
* インクリメント +=
* デクリメント -=

# ヘッダ部・スイート部

* ヘッダ部...末尾にコロンがつく
* スイート部...中身

# 標準入力

```
age_str = input("Enter your age:")
```

* 必須引数・オプション引数

	return(x)

# Range

range(5) = [0, 1, 2, 3, 4]
range(1, 5) = [1, 2, 3, 4]
range(1, 5, 2) = [1, 3]
