# 四則演算

* 累乗（るいじょう）: **  (2 ** 3 = 8)
* 割り算（切り捨て）: // (13 // 8 = 5)
* 余り: % (14 % 4)

# ビット反転

* NOT: not a
* AND: a & b
* OR: a | b
* XOR a ^ b 
* 左シフト: a << b
* 右シフト: a >> b
# 条件演算子

* a == b : a が b と等しい
* a != b : a が b と異なる
* a is b : a が b と等しい
* a is not b : a が b と異なる
* a in b : a が b に含まれる (str, list, set, tuple)
* a not in b : a が b に含まれない (str, list, set, tuple)

# 論理演算子

* a and b
* a or b
* not a
# インクリメント

* インクリメント +=
* デクリメント -=
